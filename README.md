Server setting
1. pip install -r requirements.txt

2. .env 파일 생성
   DB_USER=사용자 이름
   DB_PASSWORD=비밀번호
   DB_HOST=현재 내 호스트
   DB_PORT=포트번호
   DB_NAME=DB 이름

3. 사용자 DB 모델에 맞게 수정.

4. cmd에서 경로를 찾은 후 python app.py로 실행 추천
